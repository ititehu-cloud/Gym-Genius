# **App Name**: Gym Genius

## Core Features:

- Dashboard: Display key gym statistics at a glance (Today Date Statistics: e.g., Active members, Total Present Today, Expiry Today, Todays Payment Collection. Monthly Statistics Month Name: eg. Monthly Collection, Total Month Pending Dues. Overall Statistics: Total Collection, Total Dues)
- Member Management: Add, edit, and view member profiles in Id card format.
- Attendance Tracking: Record and monitor member attendance.
- Payment Processing: Manage member payments and generate invoices.
- Plan Management: Define and manage gym membership plans.
- Inactive Member Insights: AI tool analyzes user data and proactively identifies and informs the dashboard user when it observes indicators (change in attendance or payment patterns) that members are at risk of becoming inactive. It suggests potential interventions.

## Style Guidelines:

- Primary color: Deep teal (#468499) for a calming and trustworthy feel, conveying health and reliability.
- Background color: Soft off-white (#F0F4F5) for a clean and spacious layout.
- Accent color: Muted purple (#846694) to add a touch of sophistication and highlight key interactive elements.
- Headline font: 'Space Grotesk' sans-serif font for headings and titles; body font: 'Inter' sans-serif font for body and UI.
- Use modern, minimalist icons to represent different modules and actions.
- A clean and intuitive layout with clear navigation and well-organized data presentation.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
